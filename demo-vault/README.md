This is an [Obsidian](https://obsidian.md/) vault demonstrating features of the [Advanced Note Composer](https://github.com/mnaoumov/obsidian-advanced-note-composer) plugin.

## Opening this vault

- **If you already have the plugin installed**, run the **Advanced Note Composer: Open demo vault** command — it downloads and opens the latest demo vault automatically.
- **Otherwise**, download `advanced-note-composer.demo-vault.zip` from the [plugin releases](https://github.com/mnaoumov/obsidian-advanced-note-composer/releases), unzip it, and in Obsidian choose `Manage vaults...` → `Open folder as vault` → select the unzipped folder.

On first open, Obsidian treats the vault as untrusted — choose **Trust author and enable plugins** (or enable Community plugins manually), then open [00 Start](<./00 Start.md>).
